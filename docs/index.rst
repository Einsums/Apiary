======
Apiary
======

A libclang/libtooling-based C++ introspection engine: from a single AST walk it
emits pybind11/nanobind bindings, ``.pyi`` type stubs, Sphinx C++ API docs
(including macros — no Doxygen), and a structured public-API JSON IR.

**This site is built by apiary documenting its own headers** — a live dogfood
of its C++ documentation pipeline. The :ref:`C++ API <cpp_api>` pages below are
generated by ``apiary --emit-cpp-docs-json`` over apiary's own source.
